# 🗓️ Dia 3 -- Análise de Dados com Python

Introdução à análise de dados com Python utilizando três bibliotecas
principais responsáveis por processamento, organização e visualização de
dados.

## 📚 O Trio Essencial

-   **NumPy** → computação numérica eficiente com arrays e operações
    matemáticas
-   **Pandas** → manipulação e organização de dados em estruturas
    tabulares (DataFrame)
-   **Matplotlib** → criação de gráficos para visualização de dados

## 📌 Principais Conceitos

-   Uso de arrays multidimensionais para cálculos rápidos
-   Manipulação de dados em formato de tabela
-   Aplicação de funções estatísticas como média, soma e máximo
-   Criação de gráficos para análise visual

## 🎯 Objetivo

Compreender como trabalhar com dados em Python, passando pelas etapas de
processamento, organização e visualização.

------------------------------------------------------------------------

# 🗓️ Dia 4 -- Machine Learning com Python

No quarto dia, foi introduzido o conceito de Machine Learning aplicado à
análise de dados, focando na criação de modelos preditivos.

## 📚 Principais Conceitos

-   Diferença entre aprendizado supervisionado e não supervisionado\
-   Separação dos dados em treino e teste\
-   Criação de variáveis alvo (target)\
-   Uso de algoritmos de classificação

## 🧠 Modelos Utilizados

-   Regressão Logística\
-   Árvore de Decisão\
-   Random Forest

## ⚙️ Etapas do Processo

-   Preparação dos dados (limpeza e transformação)\
-   Codificação de variáveis categóricas\
-   Normalização dos dados\
-   Treinamento dos modelos\
-   Avaliação com métricas (ex: acurácia)

## 🎯 Objetivo

Aplicar técnicas de Machine Learning para identificar padrões e realizar
previsões a partir dos dados.
