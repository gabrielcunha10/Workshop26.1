# 🧪 Exercício: Análise de Temperaturas com Python

## 🎯 Objetivo

Simular e analisar a variação de temperaturas ao longo de uma semana utilizando Python.

---

## 📌 Descrição do Exercício

O exercício consiste em trabalhar com dados de temperatura, passando por etapas básicas da análise de dados:

### 🔹 1. Geração dos Dados

* Gerar **7 valores aleatórios** entre **20°C e 35°C**
* Criar uma lista com os **dias da semana**

---

### 🔹 2. Organização dos Dados

* Criar um **DataFrame** contendo:

  * Dias da semana
  * Temperaturas correspondentes

---

### 🔹 3. Análise dos Dados

Calcular:

* Média das temperaturas
* Temperatura máxima
* Temperatura mínima

---

### 🔹 4. Visualização

Criar **3 gráficos** para representar os dados:

* Gráfico de linha
* Gráfico de barras
* Gráfico de dispersão (ou outro tipo)

---

## 📊 Resultado Esperado

* Tabela com os dias e temperaturas
* Cálculos estatísticos realizados
* Gráficos mostrando a variação da temperatura ao longo da semana

---

## 🚀 Conclusão

O exercício demonstra, de forma prática, como:

* Gerar dados
* Organizar informações
* Analisar resultados
* Visualizar dados de maneira clara
